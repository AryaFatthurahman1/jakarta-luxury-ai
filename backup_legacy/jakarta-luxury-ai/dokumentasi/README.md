## Run Locally

**Prerequisites:**  Node.js, PHP, MySQL

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key

3. Setup database:
   ```bash
   npm run setup
   ```

4. Run the app:
   ```bash
   npm run dev
   ```
